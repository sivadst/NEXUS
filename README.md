# NEXUS 🚀

NEXUS is a smart route optimization system designed to compute efficient paths using graph-based algorithms. The project focuses on solving real-world navigation problems with optimized logic and a simple interface.

## 🔧 Features

* Optimized path finding using graph algorithms
* Backend-driven logic for route computation
* Lightweight frontend for quick interaction
* Easy to run and test locally

## 🧠 Tech Stack

* Python
* Streamlit (Backend Interface)
* HTML (Frontend view)
* Graph Algorithms (Dijkstra’s Algorithm)

## 📁 Project Structure

* `app.py` / backend files → Core logic and processing
* `nexus.html` → Static frontend interface for easy access
* `requirements.txt` → Dependencies

## 📂 How to Use

* Run the backend using Streamlit
* Open `nexus.html` for frontend interaction



1. Clone the repository:

```
git clone https://github.com/sivadst/NEXUS.git
cd NEXUS
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the backend:

```
streamlit run app.py
```

4. Open `nexus.html` in your browser for frontend view

## 📌 Note

The frontend is provided as a static HTML file to ensure easy access and quick demonstration without additional setup.
